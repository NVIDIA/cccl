# Citation Guide

## To Cite CCCL 
If you use CCCL in a publication, please use citations in the following format (BibTeX entry for LaTeX):
```tex
@Manual{,
  title = {CCCL: CUDA C++ Core Libraries},
  author = {CCCL Development Team},
  year = {2023},
  url = {https://github.com/NVIDIA/cccl},
}
```

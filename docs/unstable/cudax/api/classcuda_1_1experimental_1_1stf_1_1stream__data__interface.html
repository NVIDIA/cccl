cuda::experimental::stf::stream_data_interface#

template<typename T>
class stream_data_interface : public cuda::experimental::stf::data_impl_base<T>#

Data interface for the CUDA streams backend.

This is similar to stream_data_interface_simple but relies on a more sophisticated allocator scheme that caches memory allocations.

Subclassed by cuda::experimental::stf::slice_stream_interface< T, dimensions >

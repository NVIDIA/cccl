# cuda-cccl-experimental

Experimental CUDA Core Compute Libraries for Python.

This package provides CUDASTF (Stream Task Flow) Python bindings.

## Installation

```bash
pip install cuda-cccl-experimental[cu13]  # or [cu12] for CUDA 12.x
```

## Requirements

- `cuda-cccl` (installed automatically as a dependency)
- NVIDIA GPU with CUDA drivers
- Linux only (not supported on Windows)
